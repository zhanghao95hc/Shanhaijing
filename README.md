# Shanhaijing

This repository is for the Shanhaijing project: a Chinese mythological creature
archive inspired by descriptions in the Shan Hai Jing.

## Getting Started

Open `index.html` directly in a browser, or serve the folder locally:

```powershell
python -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173/
```

The first version includes three creatures: 九尾狐, 狰, and 狍鸮. Use the menu to
switch the whole page between creature profiles.
